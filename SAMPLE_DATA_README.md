# サンプルデータについて

## 登録されている材料

### 木材（3種類）
1. **カリン材** - 美しい木目と高い硬度
2. **栗材** - 軽量で加工しやすい
3. **樫材** - 非常に硬く、耐久性に優れる

### 金属（3種類）
4. **アルミニウム（純アルミ）** - 軽量で加工性が良い（JIS H 4000）
5. **ステンレス鋼 SUS304** - 優れた耐食性（JIS G 4305）
6. **真鍮（黄銅）** - 美しい黄金色（JIS H 3100）

### プラスチック（3種類）
7. **ポリプロピレン（PP）** - 軽量で耐薬品性に優れる（JIS K 6922）
8. **ポリエチレン（PE）** - 最も一般的な熱可塑性樹脂（JIS K 6760）
9. **ポリ塩化ビニル（PVC）** - 建築材料やパイプに使用（JIS K 6723）

## データの特徴

- **JIS規格準拠**: すべての材料がJIS規格に準拠したデータ
- **著作権フリー**: 一般的な材料情報のみを使用
- **詳細仕様対応**: レイヤー①（必須情報）を完全に記入
- **自動画像生成**: 画像がない材料は自動的にテクスチャ画像を生成

## サンプルデータの生成方法

```bash
# 依存パッケージのインストール
pip install -r requirements.txt

# サンプルデータの生成
python init_sample_data.py
```

## 画像生成機能

`image_generator.py`が以下の機能を提供します：

- **木材テクスチャ**: 木目を模したテクスチャ画像
- **金属テクスチャ**: 金属の反射を模したテクスチャ画像
- **プラスチックテクスチャ**: 滑らかなプラスチック感の画像

生成された画像は`uploads/`ディレクトリに保存され、データベースに自動登録されます。

## データの拡張

新しい材料を追加する場合は、`init_sample_data.py`を参考にしてください。
詳細仕様に準拠した形式でデータを登録することで、包括的な材料データベースを構築できます。
