# UIデザイン仕様書

## デザインコンセプト

マテリアル感のある、美しく使いやすいデータベースシステム

## 画像の使用

### 背景画像
- **メイン.webp**: アプリ全体の背景として使用
  - 固定背景（background-attachment: fixed）
  - 半透明の白いオーバーレイ（rgba(255, 255, 255, 0.85)）で可読性を確保

### 装飾画像
- **サブ.webp**: 装飾要素として使用
  - ヒーローセクションの左右に配置
  - カード内のテクスチャとして使用（低透明度）
  - 素材カードの背景テクスチャとして使用

## デザイン要素

### 1. ガラスモーフィズム（Glassmorphism）
- 半透明の背景（rgba(255, 255, 255, 0.95)）
- ぼかし効果（backdrop-filter: blur(20px)）
- 微細なボーダー（rgba(255, 255, 255, 0.8)）

### 2. マテリアルテクスチャ
- サブ.webpをテクスチャとして使用
- 低透明度（opacity: 0.03-0.1）
- mix-blend-mode: multiply で自然な合成

### 3. カードデザイン
- 角丸（border-radius: 24px）
- 多層シャドウ（複数のbox-shadow）
- ホバー時のアニメーション（transform, scale）

### 4. カラーパレット
- プライマリ: #667eea（紫）
- セカンダリ: #764ba2（濃い紫）
- アクセント: #f093fb（ピンク）
- テキスト: #2c3e50（ダークグレー）
